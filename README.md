# PPILibrary

Personal plug-in library

## 功能介绍

- [x] `Core` 核心基础扩展， 不依赖于任何三方库

## 使用

- 根据`subspec`指定

    ```ruby
    pod 'PPILibrary', :subspecs => ['Core'], :git => 'https://github.com/ShenYj/PPILibrary.git'
    ```

- 指定版本

    ```ruby
    pod 'PPILibrary', :git => 'https://github.com/ShenYj/PPILibrary.git', :tag => '0.0.2'
    ```
